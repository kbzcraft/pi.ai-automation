# pi.ai_automation

create a .auth dir and pi.json file fill id and pass and ready to go

To set up project

In Linux:

Create a virtual environment 
```bash
python -m venv .venv
```
Activate the environment 
```bash
source .venv/bin/activate
```
Install packages
```bash
pip install playwright
```
```bash
playwright install
```
```bash
mkdir .auth
```
```bash
touch .auth/pi.json
```
